# 주식예약주문정정취소

## API Overview

**API ID**: `v1_국내주식-018,019`
**Category**: [국내주식] 주문/계좌
**HTTP Method**: POST
**URL**: `/uapi/domestic-stock/v1/trading/order-resv-rvsecncl`
**Communication Type**: REST

## Description

주식예약주문정정취소 API for [국내주식] 주문/계좌.

## TR_ID Information

| Environment | TR_ID |
|-------------|-------|
| Real Trading | `(예약취소) CTSC0009U (예약정정) CTSC0013U` |
| Paper Trading | `Not Supported` |

**Note**: Paper trading is not supported for this API.

## Endpoint

```
POST /uapi/domestic-stock/v1/trading/order-resv-rvsecncl
```

## Request Headers

| Header | Value | Required | Description |
|--------|-------|----------|-------------|
| `Authorization` | `Bearer {access_token}` | Yes | OAuth 2.0 access token |
| `appkey` | `{app_key}` | Yes | KIS OpenAPI app key |
| `appsecret` | `{app_secret}` | Yes | KIS OpenAPI app secret |
| `tr_id` | `(예약취소) CTSC0009U (예약정정) CTSC0013U` | Conditional | TR_ID for real trading |
| `tr_id` | `N/A` | Conditional | TR_ID for paper trading |
| `custtype` | `P` or `B` | Yes | Customer type (P: Individual, B: Corporate) |


## Request Parameters

### Required Parameters

| Parameter | Type | Description | Required |
|-----------|------|-------------|----------|
| `tr_id` | string | TR_ID (Real: `(예약취소) CTSC0009U (예약정정) CTSC0013U`, Paper: `모의투자 미지원`) | Yes |
| `CANO` | string | Account number (8 digits) | Conditional |
| `ACNT_PRDT_CD` | string | Account product code (2 digits) | Conditional |
| `ETC_TR_ID_PREFIX` | string | TR_ID prefix for multiple TR_IDs | Conditional |
